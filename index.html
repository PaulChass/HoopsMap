Informations

Hoops Map est une application qui vous permet de trouver un playground de basket à côté de chez vous.

Vous pouvez rechercher un playground en entrant une ville ou un code postal dans la barre de recherche ou en cliquant sur le bouton "Localiser" pour trouver les playgrounds à proximité de votre position actuelle.

Les données sont fournies par la base de données du Ministère des Sports et sont mises à jour régulièrement par cette structure.

Si vous souhaitez ajouter un playground, vous pouvez le faire ici.
